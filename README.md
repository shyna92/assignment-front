# Technical Assessment: React and CSS

This assessment tests your ability to identify and fix common React and CSS issues.

## Tasks

1. Fix what issues you can find in the UserCard component and the ProductList.scss file
    - Consider React best practices and accessibility guidelines like WCAG 2.1
2. Improve the ProductList layout using flexbox:
   - Products should wrap to next line on smaller screens
   - Each product should take up 33.33% of the container width (minus gap)

## Getting Started

1. Unzip this repository
2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`
4. Fix the issues
5. Submit your solution.  A zip file of the repo(minus node_modules, .next, and .git) is sufficient.
